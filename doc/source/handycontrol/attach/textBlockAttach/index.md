# 属性

| 名称 | 用途 |
|-|-|
| AutoTooltip | 设置是否自动显示工具提示 |

# 使用案例

## AutoTooltip 设置是否自动显示工具提示

```xml
<hc:SimplePanel Width="100" VerticalAlignment="Center" HorizontalAlignment="Center">
    <TextBlock hc:TextBlockAttach.AutoTooltip="True" Text="EEEEEEEEEEEEEEEEEEEE"/>
</hc:SimplePanel>
```
